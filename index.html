Core Theory
SPOF Single Point of Failure — one node whose death kills the entire system. In a centralized fleet of N drones, the ground station is a SPOF.
O(N) Complexity Centralized comms scale as O(N) — doubling the fleet doubles the controller's load. Decentralized scales as O(k) where k = fixed neighbor count.
CAP Theorem A distributed system cannot simultaneously guarantee Consistency, Availability, and Partition tolerance. Centralized fleets choose CA. Decentralized swarms choose AP.
Latency Centralized round-trip = sensor→controller→command → 100–500ms. Decentralized local reaction = sensor→act → <10ms.

Boids Algorithm — Reynolds 1987
Separation steer += (self.pos − neighbor.pos).normalize() for neighbors within sepRadius. Prevents collision.
Alignment steer += (avg_neighbor_velocity − self.velocity) × aliWeight. Creates coherent heading.
Cohesion steer += (centroid_of_neighbors − self.pos) × cohWeight. Gravitational pull of the group.
Emergence No drone knows the formation shape. The shape emerges from N agents each running 3 rules locally.
Topological vs Metric Metric: sense all neighbors within radius R. Topological: sense exactly k nearest neighbors regardless of distance. Starlings use k=7.
Noise Calibrated randomness prevents swarms getting stuck in local optima. Too little = brittle. Too much = chaos.

Mesh Networks & Gossip Protocol
Inverse Square Law Signal power ∝ 1/d². Double the distance → quarter the signal. At 2.4GHz, -10 to -20dB loss per concrete wall.
Gossip Convergence In a swarm of N drones each gossiping to k neighbors, message reaches all nodes in log_k(N) cycles. N=100, k=3 → 4.2 cycles ≈ 420ms.
Latency Stack sensor(1-5ms) + decision(1-10ms) + encode(0.1ms) + TX(0.5ms/hop) × hops = 30-80ms end-to-end for 6 hops.
TDMA Time Division Multiple Access — each drone gets an assigned time slot to transmit, eliminating simultaneous collision.
Hidden Node Node A and C both in range of B but not each other. Both transmit to B simultaneously → packet collision.
Quorum A detection requires N independent confirmations before action. Prevents false positives from propagating.

Path Planning & Obstacle Avoidance
Potential Fields F_total = W_goal × attract(goal) + W_wall × repel(walls) + W_agent × repel(agents). Each force = strength × direction_vector.
Local Minima A point where ∇F = 0 and F > 0. Forces balance to zero net force. Drone oscillates permanently. Only Layer 3 (A*) escapes this.
A* Algorithm f(n) = g(n) + h(n). g = cost from start. h = heuristic to goal (Manhattan distance). Always expand lowest-f frontier node.
SLAM Simultaneous Localization And Mapping. Chicken-and-egg: need map to localize, need position to map. Solved with probability distributions.
Frontier Exploration Frontier = explored cell adjacent to unexplored cell. Assign each drone its nearest unclaimed frontier. Map grows outward until no frontiers remain.
Stigmergy Indirect coordination through environment modification. Ants leave pheromones. Drones leave digital marks. Other agents' behavior changes in response.
Velocity Obstacles VO(A,B) = set of velocities for A that cause collision with B within time T. Agent selects velocity outside all VO cones.

Emergence ALL

Swarm Intelligence ALL

Collective behavior of decentralized, self-organized systems following local rules.

Boids M2

Craig Reynolds' 1987 algorithm simulating flocking using three rules: separation, alignment, cohesion.

Separation M2

The rule that steers a drone away from crowded neighbors to prevent collision.

Alignment M2

The rule that steers a drone toward the average heading of its neighbors.

Cohesion M2

The rule that steers a drone toward the centroid of its neighbor group.

Topological Neighbor M2

A neighbor defined by social rank (e.g., the 7 nearest) regardless of physical distance.

Metric Neighbor M2

A neighbor defined by physical distance — all agents within radius R.

Local Minima M4

A point in a potential field where all forces balance to zero, trapping an agent permanently.

Potential Fields M4

Navigation method where obstacles repel and goals attract, producing a force vector per agent per frame.

A* M4

Pathfinding algorithm using f(n)=g(n)+h(n) to find optimal paths on a known map.

SLAM M4

Simultaneous Localization And Mapping — building a map while tracking position within it.

Frontier Exploration M4

Strategy where drones target the boundary between explored and unexplored space.

Stigmergy M4

Indirect coordination where agents modify the environment and others respond to those modifications.

Velocity Obstacle M4

The set of velocities that would cause an agent to collide with a neighbor within time T.

Deadlock M4

A stable equilibrium where two agents block each other permanently with no self-resolving mechanism.

SPOF M1

Single Point of Failure — one node whose death causes total system failure.

Decentralized Control M1

Architecture where each agent makes decisions locally with no central coordinator.

CAP Theorem M1

A distributed system cannot simultaneously guarantee Consistency, Availability, and Partition tolerance.

Mesh Network M3

A network where every node is simultaneously sender, receiver, and router — no infrastructure dependency.

Gossip Protocol M3

Information spreading method where each node shares state with k random neighbors each cycle.

Quorum M3

Minimum number of independent confirmations required before a detection is acted upon.

TDMA M3

Time Division Multiple Access — each node gets an assigned transmit slot, eliminating collision.

Hidden Node Problem M3

Two nodes both in range of a third but not each other, causing simultaneous collision at the third.

Attenuation M3

Signal power loss with distance following inverse square law: P ∝ 1/d².

Latency M3

Time delay between event occurrence and action taken in response.

Self-Healing M1

Property of mesh networks to automatically reroute around failed nodes.

GPS-Denied Navigation M3

Operating without GPS using visual odometry, IMU dead reckoning, or UWB ranging.

Cooperative SLAM M4

Multiple agents sharing sensor observations to build a collective map faster than any individual could.

Emergent Behavior ALL

Behavior that was never explicitly programmed, arising only from local agent interactions.
